## ⚡ Quick Start

### Prerequisites
- **Node.js** (v16.0.0 or higher)
- **npm** or **yarn** package manager
- **Git** for version control

### One-Command Setup
```bash
# Clone and setup in one go
git clone https://github.com/Ihsas01/React-Portfolio.git && cd React-Portfolio && npm install && npm run dev
```

## 📦 Installation

### Step 1: Clone the Repository
```bash
git clone https://github.com/Ihsas01/React-Portfolio.git
cd React-Portfolio
```

### Step 2: Install Dependencies
```bash
npm install
# or
yarn install
```

### Step 3: Start Development Server
```bash
npm run dev
# or
yarn dev
```

### Step 4: Open in Browser
Navigate to `http://localhost:5173` to view your portfolio locally.

## 🌐 Deployment

### GitHub Pages (Current)
The portfolio is automatically deployed to GitHub Pages using the `gh-pages` package.

```bash
# Deploy to GitHub Pages
npm run deploy
```

### Alternative Deployment Options

#### Vercel
```bash
# Install Vercel CLI
npm i -g vercel

# Deploy to Vercel
vercel
```

#### Netlify
```bash
# Build the project
npm run build

# Deploy to Netlify (drag dist folder to Netlify dashboard)
```
